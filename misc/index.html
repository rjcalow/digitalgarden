Breville/sage barrista pro grinder

Disassemble grinder

Remove the bean hopper and beans.

Pull up the handle on the top conical bur. Unlock by rotating it left.

Clean with a hoover. Remove the reverse nut and washer. The bur can now be lifted out.

The second washer and bottom bur can be lifted with pillars. Underneath is a delicate felt washer. These can be replaced.

Xiaomi Mijia LYWSD03MMC

About

A cheap Bluetooth Low Energy (BLE) device that broadcasts and displays temperature and humidity.

I did not pair the units with the Xiaomi app, but used python instead.

Aim

I wanted the ability to check and record temperature and humidity in my DIY mini greenhouse.

Python

Using the lywsd03mmc package, I can retrieve data without the need to reflash the units.

Real time scraping or retrieve history?

I wrote a short script that would connect to the LYWSD03MMC and upload data to a google sheet.

This is problematic as the signal is sketchy with distances. It wouldn't always connect. I had the script loop until the returning object has a temperature attribute. 

while True:
    print("connecting...")
    try: client = connect(greenhouse)
    except: pass
    if hasattr(client, 'temp'):
        break

Another long term problem is battery life. Connecting at regular intervals will drain the battery. This is something the atc firmware addresses.

Another option is to retrieve history. The lywsd03mmc package has a tool for exporting this data to csv format.

lywsd03mmc2csv A4:C1:38:12:34:56 --output data.csv

The format has a max and min for each hour.

Time Min temperature Min humidity Max temperature Max humidity
2022-01-31 13:15:52.717247 15.1 36 19.1 61
2022-01-31 14:15:52.717247 12 51 20 66

Possible applications

  • Telegram bot to alert if the greenhouse gets too hot in summer.

  • Automate opening the greenhouse door/window.

  • Automate watering if very hot.

  • Compare records with weather forecast and gauge the efficiency of the greenhouse.
