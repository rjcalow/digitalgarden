spin down usb hdd

Raspberry pi(x) will constantly run usb hard drives.

Use hdparm (hard disk parameter) tool.

On libreelec 9.2.

$ nano /storage/.config/autostart.sh

Enter:

(sleep 60; hdparm -B 127 -M 254 -S 60 /dev/sda1 ) &

-B 127 allows spin down

Possible settings range from values 1 through 127 (which permit spin-down), and values 128 through 254 (which do not permit spin-down). [source]

-M 254 sets the acoustic management

128 is the most quiet (and therefore slowest) setting and 254 the fastest (and loudest). [source]

-S 60 puts the drive into idle low power mode with a timeout value

Values from 1 to 240 specify multiples of 5 seconds, yielding timeouts from 5 seconds to 20 minutes. [source]

/dev/sda1 points to the mount point. Depending on os mount point naming system.
